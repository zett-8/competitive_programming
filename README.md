# Python tips for competitive programming

## Input & Output
#### Multiple inputs on a line
```python
2 4 5 7
```
```python
a, b, c, d = map(int, input().split())
# a=2 b=4 c=5 d =7

a = list(map(int, input().split()))
# a = [2, 4, 5, 6] 
```
#### Input on multiple lines
```python
3  # n number of A
22 # A1 
14 # A2
45 # A3
```
```python
n = int(input()) 
t = [int(input()) for _ in range(n)] 
```

#### Multiple inputs on multiple line
```python
4
22 1 45
25 53 110
4 444 555
2 5 0
```
```python
n = int(input())
e = [list(map(int, input().split())) for x in range(n)] 
#  [[22, 1, 45], [25, 53, 110], [4, 444, 555], [2, 5, 0]]
```
## Speeding Up
#### Don't use 'del', 'insert'
#### Use 'tuple'
#### 'sort()' is bit faster than 'sorted()'
#### Put smaller loop outside
```python
for i in range(n):
    for j in range(2):
        a += 1
        
# 5286 ms
```
```python
for i in range(2):
    for j in range(n):
        a += 1 
        
# 1923ms
```

#### Calculate combination (select r from n)
```python
from operator import mul
from functools import reduce

def cmb(n, r):
    r = min(n - r, r)

    if r <= 0:
        return 1

    over = reduce(mul, range(n, n - r, -1))
    under = reduce(mul, range(1, r + 1))
    return over // under
```
